#### 바닐라 JavaScript를 사용한 이유

[DOM을 깨우치다](https://product.kyobobook.co.kr/detail/S000001556099)를 읽고 DOM 라이브러리 없이 스크립트를 작성하여 HTML을 효율적으로 조작하는 방법을 배웠다. 책을 통해 익힌 이론을 바탕으로 직접 구현하며 내 지식으로 만들기 위해 JavaScript를 사용하였다.
